Not everything is documented and not everything needs to be documented. For example things like
fnm sourcing in `~/.config/fish/conf.d/fnm.fish` or the auto generated fnm completions are not documented.

# Themes
- [Catppuccin Macchiato](https://github.com/catppuccin/fish/blob/main/themes/Catppuccin%20Macchiato.theme)
# Functions
- `~/.config/fish/functions/cdp.fish` provides an interactive way to navigate through the shallow levels of the projects directory.
# Utilities
- `~/.config/fish/conf.d/ghostty-ssh-integration-wrapper.fish` is a SSH function wrapper, on top of the ghostty integration, that changes the color scheme of the terminal, when connecting via ssh.
